# 프로젝트 목표 및 기대효과
### 목표
+ 화상회의 참여자들의 얼굴을 촬영. 이를 학습시켜 각 참여자들의 신분을 나타낸다.

### 기대효과
+ 출석에 활용하는 시간 감소 
+ 실시간 촬영을 통한 회의 참여 유도 및 효율향상
+ 원격회의 서비스 활용도 증가
